\section{Contexte}
La prise de parole en public est une compétence essentielle, mais le feedback personnalisé est coûteux et rare. Le projet SpeechCoach vise à démocratiser ce coaching grâce à l'intelligence artificielle multimodale.

\section{Objectifs du PFE}
L'objectif est de développer un prototype logiciel capable de :
\begin{itemize}
    \item Ingérer une vidéo de présentation.
    \item Analyser l'audio (transcription, débit, hésitations).
    \item Analyser la vidéo (présence visage, regard, qualité).
    \item Fournir un rapport pédagogique automatisé.
\end{itemize}

\section{Périmètre (Scope)}
Le projet se concentre sur le traitement \textbf{offline} (différé) de vidéos courtes (2-5 minutes) avec un seul locuteur face caméra. Le système est conçu pour tourner sur un ordinateur standard (CPU friendly si possible).
